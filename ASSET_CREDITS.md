# Asset Credits / Notes

This build includes a custom TCG-style asset pack generated for Werewolf by Ryuu:

- `public/assets/tcg/templates/` — TCG-style rarity frames.
- `public/assets/tcg/roles/` — role cards for Werewolf, Villager, Seer, Doctor, Hunter, Witch, and other roles.
- `public/assets/tcg/skins/` — cosmetic skin cards.
- `public/assets/tcg/effects/` — skill/event effect cards.
- `public/assets/tcg/asset-manifest.json` — path mapping for future integration.

The card layout is designed to match the TCG frame style requested by the project. The OpenGameArt TCG Templates page by KillGorack lists the template pack as CC0 and describes it as rendered frames for trading-card-game concepts. The exact 104 MB source pack is not bundled here to keep the deploy ZIP small; this game includes lightweight SVG assets compatible with Railway and mobile browsers.

Recommended external source page:
https://opengameart.org/content/tcg-templates

Other asset ideas to manually add later:
- OpenGameArt spell/skill icon collections
- CC0 character packs from OpenGameArt
- LPC werewolf/villager sprites if you accept CC-BY-SA attribution requirements

For production, always check each asset license before redistributing.
